# 297. Serializar e Desserializar Árvore Binária

Dado o nó raiz `root` de uma árvore binária, implemente os métodos para serializar a árvore em uma string e, em seguida, desserializar essa string de volta para a árvore original.

O formato da serialização utilizado aqui é uma DFS em pré‑ordem, separando valores por vírgulas e usando `N` para nós nulos.

## Exemplo

Entrada: root = [1,2,3,null,null,4,5]

Serialização: "1,2,N,N,3,4,N,N,5,N,N"

Após desserializar a string, a árvore reconstruída corresponde à original.

## Abordagem

- Pré‑ordem (raiz, esquerda, direita):
  - Ao visitar um nó nulo, gravar `N`.
  - Caso contrário, gravar o valor e continuar recursivamente para esquerda e direita.
- Para desserializar, ler os tokens sequencialmente e construir a árvore recursivamente:
  - Se o token for `N`, retornar `null`.
  - Caso contrário, criar um nó com o valor e construir seus filhos esquerdo e direito na sequência.

## Complexidade

- Tempo: O(n), visitamos cada nó exatamente uma vez.
- Espaço: O(n) para a lista de tokens e para a pilha de recursão em árvores degeneradas.

## Restrições

- Número de nós no intervalo [0, 10^4].
- Valores dos nós dentro do domínio de inteiros.
