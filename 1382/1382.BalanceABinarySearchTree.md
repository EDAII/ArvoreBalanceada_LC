# 1382. Balancear uma Árvore de Busca Binária

Dado o nó raiz `root` de uma **árvore binária de busca**, retorne uma **árvore binária de busca balanceada** com os mesmos valores de nó.  
Se houver mais de uma resposta possível, retorne qualquer uma delas.

> Uma árvore de busca binária é **balanceada** se a profundidade das duas subárvores de cada nó nunca diferir em mais de 1.

## Exemplo 1

**Entrada:** raiz = [1,null,2,null,3,null,4,null,null]

**Saída:** [2,1,3,null,null,null,4]

**Explicação:** Esta não é a única resposta correta, [3,1,4,null,2] também está correta.

##  Restrições

- O número de nós na árvore está no intervalo **[1, 10⁴]**  
- `1 <= Node.val <= 10⁵`
